📜 خطوات التثبيت اليدوي (CLI)

1. ✅ أنشئ ملف السكربت
nano ultrashield-installer.sh

2. 🔥 استبدل المتغير $GAME_PORT
GAME_PORT="27015" (قم بذلك في كل مكان داخل السكربت وداخل كود Python)

مثال:
GAME_PORT = "27015"

3. ✅ اجعل السكربت قابلًا للتنفيذ
chmod +x ultrashield-installer.sh

4. 🚀 شغل السكربت كـ root
sudo ./ultrashield-installer.sh

5. ✅ تأكد من صلاحيات الملفات الأخرى
sudo chmod +x /usr/local/bin/ai_packet_analyzer.py
sudo chmod +x /usr/local/bin/send_attack_alert.sh

6. 🧠 تحقق من تشغيل البوت والتحليلات
ps aux | grep ai_packet_analyzer
أو
 tail -f /var/log/syslog

7. 🧪 افحص iptables
sudo iptables -L ULTRA_SHIELD -v -n

8. 🔁 تشغيل دائم عند إعادة التشغيل
استخدم pm2 أو systemd
